# Проект автоматизации ООО «Новый Горизонт» 🚀

**Выполнил:** студент группы К520с11-2 Романом

## О проекте
Разработка прототипа веб-ресурса для ивент-агентства «Новый Горизонт» (организация тимбилдингов и корпоративов). Цель проекта — автоматизировать процесс сбора первичных заявок от клиентов и сократить время на расчет сметы.

## Структура репозитория
* `/Analytics` — Модели бизнес-процессов (AS IS / TO BE) в нотации BPMN 2.0.
* `/Prototypes` — Вайрфреймы ключевых экранов и иерархическая структура (Сайтмэп).
* `/Docs` — Сценарии и отчет о юзабилити-тестировании (Этап 3).
